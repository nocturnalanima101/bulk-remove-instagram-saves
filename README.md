This is a script to bulk unsave all instagram saved posts/reels.
A Playwright-based automation script that removes saved Instagram posts in bulk while keeping track of progress between runs.
No passward is stored in code as it uses existing chrome profile where you logged in.
Behaves like a human while scrolling and deleting with realistic delays.

The script will:

Open Instagram.
Navigate to your Saved Posts.
Find saved posts and reels.
Remove saves one by one.
Store progress automatically.
Resume from the last processed post if interrupted.

Notes
Instagram UI changes may require selector updates.
The script relies on the current Instagram web interface.
